[컴퓨터비전 실습]

1. 간판 사진 수집 : 2~3개 정도
2. openCV 와 Tesseract 사용
 - 소스 파일명 : sign.py (함수기반으로 작성)
	소스 파일 아래쪽에 main 에서 함수 실행하도록 작성함
 	=> exe 파일로 만들기함
 - 간판 이미지에서 글자 추출까지 코드 작성
 - 추출한 글자에서 상호명(name)과 전화번호(tel)를 딕셔너리에 저장함
 - 오라클 수업용계정에 테이블 만들고 딕셔너리 정보 저장 처리함

3. 소스파일은 vision_본인이름.zip 으로 압축해서 제출함
 - 파이참 프로젝트명 : vision
 - vision/dbscript 폴더 만들기 : vision.sql 로 테이블 만들기 구문 작성 저장함
4. 제출 ppt 에 실행 출력 화면 캡쳐해서 ppt에 추가해서 pdf 로 저장해서 제출함
 => 3개 파일 제출임 (zip, pdf, exe)
